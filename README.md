### Build e deploy

Entrar na pasta docker

```
$ cd docker
```


Rodar docker compose 

```
$ docker-compose --build
```

Matar instancia do compose

```
$ docker-compose down
```

Matar instancia do compose e deletar volumes

```
$ docker-compose down -v
```
